***************************************
*** Portfolio Plugin by Theme Honey ***
***************************************

Welcome! Here you will find the documentation for the Portfolio Plugin by Theme Honey. If you'd like to display a portfolio on your WordPress website, you must install this plugin. Upon installation, a menu item called "Portfolio" will appear below the menu item "Posts".
